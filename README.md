# TDC 2015 - Palestra AngularJS e Material Design
Aplicação de exemplo da palestra no TDC 2015 em Florianópolis.

**Requisitos:**
  - NodeJS: https://nodejs.org/
  - Bower: ```npm install -g bower```

**Instalar pacotes:**
```javascript
npm install && bower install
```

**Rodar a demo**
```javascript
gulp serve
```
